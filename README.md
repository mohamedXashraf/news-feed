# News Feed

- Simple Android App for displaying News.

![](https://github.com/mohamedXashraf/news-feed/blob/master/banner.png?raw=true)

- Topics:
  * Modularization.
  * Clean Architecture: 3 Modules : (Presentation, Domain, Data) Layers.
  * Presentation Layer:
    * Jetpack Compose.
    * View-Model.
    * MVI Presentation Architecture Pattern.
    * MVVM Presentation Architecture Pattern.
  * Domain Layer:
    * UseCases.
    * Repository Interfaces.
  * Data Layer:
    * Repository Implementation.
    * Models.
    * APIs: Retrofit.
  * Koin (Dependency Injection).
  * Kotlin Coroutines.
